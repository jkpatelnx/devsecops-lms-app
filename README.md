## 🚀 Deployment
This app is containerized and deployed via a full DevSecOps CI/CD pipeline:  
👉 [devsecops-lms-deployment](https://github.com/jkpatelnx/devsecops-lms-deployment)

## 🛠️ Tech Stack
- **App:** Moodle v5.1.0 (PHP 8.3, Nginx, PHP-FPM)
- **Database:** MariaDB 10.11
- **Container:** Docker + Docker Compose
- **Cloud:** AWS EC2
- **CI/CD:** GitHub Actions with Trivy, Gitleaks, Hadolint
